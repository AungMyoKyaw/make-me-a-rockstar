# [ MAKE-ME-A-ROCKSTAR ][make-me-a-rockstar]

> Make you a rockstar polyglot developer

## Install

```shell
npm install -g make-me-a-rockstar
```

## Usage

```shell
Usage
  $ rockstar
Options
  --polyglot
  --lang
  --day
  --month
  --year
  --email
  --name
Examples
  $ rockstar
  $ rockstar --lang=javascript --day=1 --month=1 --year=1 --email=user@example.com --name='Mr Test'
```
