# MAKE-ME-A-ROCKSTAR

> Make you a rockstar polyglot developer

## Install

```shell
npm install
```

## Usage

```shell
Usage
  $ rockstar <input>
Options
  --polyglot
  --lang
  --day
  --month
  --year
  --email
  --name
Examples
  $ rockstar
  $ rockstar --lang=javascript --day=1 --month=1 --year=1 --email=user@example.com --name='Mr Test'
```
